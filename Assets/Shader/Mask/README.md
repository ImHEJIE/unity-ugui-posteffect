# mask-effect

> 概要

マスク画像を画面に加算します。<br>
カラーのRを加算度に割り当てているので、徐々に画面が隠れていくといったアニメーションを作ることもできます。<br>
また、テクスチャを差し替えることで様々なフェードイン/アウトの演出を作ることができます。<br>
